fish-in-txt

% 説明

fish-in-txtはシンプルなテキストエディタです。ちょっとしたメモを書く時のために作られました。

"#"を用いてインデックスを自動生成できます。機能は実装できたタイミングで追加しています。

シンプルでかわいいを維持しながら、機能を追加し、非シンプルとシンプルの狭間を彷徨います。

「mikan

「coffe

「kawaii icons

% 機能

-- #:主題

-- ##:副題

-- window:最大化/最小化/閉じる

-- 保存:新規保存/上書き保存

-- drug&drop:ファイル読み込み

-- ctrl+s:保存

-- windowサイズ:前回のサイズの復元

-- theme:カラーとフォントサイズの変更

% theme.json

userDataに"theme.js"を入れることでテーマカラーを変更できます。

{
"main": "#FFFFFF",
"sub" : "#E47830",
"text": "#FC5855"
}

% config.json

userDataの中に自動で生成されるファイルです。アプリを閉じた際に、windowサイズが記録されます。

font-sizeを書き換えることでアプリケーションでのフォントサイズを変更できます。
