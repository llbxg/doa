fish-in-txt

fish-in-txt is a simple text editor.

This app was created to write articles for this website.

While keeping it simple and cute, it adds functions and wanders between non-simple and simple.

The index can be automatically generated using "#".

% functions

Functions are added when they can be implemented.

  • #: Title
  • ##: Sub title
  • window: Maximize/Minimize/Close
  • Save: New save / Overwrite save
  • Drug&Drop: File reading
  • Ctrl+S: Save
  • Window size: Restore previous size
  • Theme: Change color and font size
